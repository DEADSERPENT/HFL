━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

The model must satisfy four hard constraints simultaneously:

  Constraint 1: ONNX inference < 100ms on edge hardware
    → Eliminates: ViT, ResNet-50, EfficientNet-B7, BERT, GPT-variants
    → Valid: MobileNetV3, EfficientNet-B0, TinyBERT, 1D-CNN, GRU

  Constraint 2: DP-SGD compatible (gradient clipping per sample)
    → Requires: no BatchNorm (BN sensitive to per-sample clipping)
    → Solution: Replace BatchNorm with GroupNorm or LayerNorm
    → Valid: Any model after BN→GN substitution

  Constraint 3: Multimodal (2 input modalities, heterogeneous)
    → Requires: dual-branch encoder + fusion module
    → Must handle missing modality gracefully (sensor failure case)

  Constraint 4: FL-compatible (local training on limited data)
    → Small model prevents overfitting on device's limited local data
    → <3M parameters preferred (reduces communication overhead)

  SELECTED ARCHITECTURE — MODEL 1: HFL-MM (Phase 5, Primary)
  ════════════════════════════════════════════════════════════
  Encoder A: 1D-CNN + GRU      (time-series, vibration, EEG, audio features)
  Encoder B: MobileNetV3-Small  (images: spectrograms, X-rays, camera)
  Fusion   : Late Fusion FC     (concatenate 128-d + 576-d → 256-d)
  Training : DP-SGD via Opacus  (GroupNorm replaces BatchNorm)
  Deployment: ONNX Runtime INT8 (edge inference)
  Params   : ~3.0M | ONNX INT8: ~2.1 MB | P95 latency: ~68ms

  ADVANCED ARCHITECTURE — MODEL 2: PHANTOM-FL (Extended Contribution)
  ════════════════════════════════════════════════════════════════════
  Tokenizer: Unified Patch Tokenizer (signal patches + image patches → D=128)
  Encoder  : Cross-Modal Gated Attention (CMGA, 3 blocks, 8 heads)
  Privacy  : Sensitivity-Adaptive DP (SADP, per-layer σ_l)
  FL Agg.  : Federated Knowledge Distillation at Edge (FedKD-E)
  Sync     : Asynchronous Staleness-Weighted Aggregation (ASWA)
  Personal : Per-device Adapter Layer (pFAL, 16K params, never shared)
  Deployment: ONNX Runtime INT8 (attention opset 17 supported)
  Params   : ~4.2M backbone + 16K adapter | ONNX INT8: ~2.8 MB | P95: ~75ms

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

2.1  Option A: Pure Transformer (e.g., small ViT + Audio Spectrogram Transformer)
  Pros: State-of-the-art accuracy on multimodal tasks, natural attention fusion
  Cons:
    - Inference latency: ~200–500ms on edge (ViT-Tiny: 4.7M params, 1.3B FLOPs)
    - DP-SGD with attention is unstable (attention softmax + clipping = gradient explosion)
    - Requires large local datasets for self-attention to generalize (fails at FL scale)
  VERDICT: Rejected — violates <100ms latency constraint

2.2  Option B: EfficientNet-B0 (both modalities, shared backbone)
  Pros: Excellent accuracy-efficiency trade-off (5.3M params, 390M FLOPs)
  Cons:
    - Images only — cannot process raw time-series signal (CWRU, WESAD)
    - Would require converting all signals to images (information loss)
    - BatchNorm throughout (poor DP-SGD compatibility)
  VERDICT: Rejected — not native time-series support

2.3  Option C: ResNet-18 + LSTM (two separate full models)
  Pros: Well-understood, good accuracy
  Cons:
    - ResNet-18: 11.7M params × 2 = 23.4M total → 93.6 MB model
    - ONNX inference: ~130ms on edge
    - Too large for ONNX INT8 quantization under 5MB constraint
  VERDICT: Rejected — latency and model size constraints

2.4  Option D: TinyML approach (MCUNet / MobileNetV2)
  Pros: <300KB model, runs on microcontrollers, <10ms inference
  Cons:
    - Accuracy drops ~15–20% vs. MobileNetV3 on our tasks
    - Insufficient for multimodal fusion (no capacity for late fusion head)
    - Not suitable for FL convergence with non-IID data
  VERDICT: Rejected — insufficient accuracy

2.5  SELECTED (Phase 5): HFL-MM (1D-CNN+GRU + MobileNetV3-Small + Late Fusion)
  Pros:
    - 1D-CNN+GRU: 420K params, native time-series, GRU < LSTM in memory
    - MobileNetV3-Small: 2.5M params, 56M FLOPs, designed for mobile edge
    - GroupNorm: replaces BatchNorm for DP-SGD compatibility
    - Late fusion: privacy-optimal (DP applied at fused gradient level)
    - ONNX INT8: ~68ms P95 latency on RTX A2000 (edge proxy)
    - Total: ~3.0M params, ~8.4MB FP32, ~2.1MB INT8
  VERDICT: SELECTED for Phase 5/6 — all constraints satisfied, low impl. risk

2.6  ADVANCED (Extended): PHANTOM-FL (Patch Tokenizer + CMGA + SADP + FedKD-E)
  Pros:
    - Cross-Modal Gated Attention (CMGA): modalities inform each other
      during encoding — signal tokens attend to image tokens and vice versa
    - Sensitivity-Adaptive DP (SADP): per-layer σ_l reduces accuracy loss
      from ~1.5% (HFL-MM) to ~0.8% with same total ε=1.0 budget
    - FedKD-E: distillation at edge avoids gradient conflict under non-IID
    - ASWA: no straggler bottleneck — aggregates when 70% devices respond
    - pFAL: +2–4% per-device accuracy via 64KB local adapter (no DP needed)
    - Total backbone: ~4.2M params | ONNX INT8: ~2.8 MB | P95: ~75ms
    - 5 independently publishable contributions (vs HFL-MM's 4)
  Cons:
    - ~3–4 weeks more implementation time vs HFL-MM
    - ONNX attention export requires sdpa_kernel override (solvable)
    - FedKD-E needs a small proxy dataset at each edge server
  VERDICT: RECOMMENDED for maximum journal novelty — implement after HFL-MM
           or as direct replacement if timeline allows (+4 weeks)

  │ NOTE: GroupNorm replaces ALL BatchNorm layers throughout.        │
  │ This is MANDATORY for Opacus DP-SGD compatibility.              │
  │ Opacus will raise an error if BatchNorm layers are present.     │

  class Encoder_A_1DCNN_GRU(nn.Module):
  """
  Input: (B, C_in, L) where C_in=1 (vibration) or 5 (WESAD 5-channel)
  Output: (B, 128) feature vector
  """
      def __init__(self, in_channels=1, gru_hidden=128):
          super().__init__()
          self.conv1 = nn.Sequential(
              nn.Conv1d(in_channels, 32, kernel_size=7, padding=3),
              nn.GroupNorm(8, 32),      # 8 groups of 4 channels
              nn.ReLU(),
              nn.MaxPool1d(4)
          )
          self.conv2 = nn.Sequential(
              nn.Conv1d(32, 64, kernel_size=5, padding=2),
              nn.GroupNorm(8, 64),
              nn.ReLU(),
              nn.MaxPool1d(4)
          )
          self.conv3 = nn.Sequential(
              nn.Conv1d(64, 128, kernel_size=3, padding=1),
              nn.GroupNorm(8, 128),
              nn.ReLU(),
              nn.MaxPool1d(4)
          )
          self.gru = nn.GRU(
              input_size=128, hidden_size=gru_hidden,
              num_layers=2, batch_first=True,
              dropout=0.2
          )

      def forward(self, x):
          # x: (B, C_in, L=1024)
          x = self.conv1(x)     # (B, 32, 256)
          x = self.conv2(x)     # (B, 64, 64)
          x = self.conv3(x)     # (B, 128, 16)
          x = x.permute(0, 2, 1)  # (B, 16, 128) for GRU
          _, h_n = self.gru(x)    # h_n: (2, B, 128)
          return h_n[-1]          # (B, 128) last GRU hidden state

  class Encoder_B_MobileNetV3(nn.Module):
  """
  Input: (B, 3, 224, 224) — spectrogram or X-ray image
  Output: (B, 576) feature vector
  """
      def __init__(self, pretrained=True, freeze_backbone=True):
          super().__init__()
          import torchvision.models as models
          backbone = models.mobilenet_v3_small(
              weights='IMAGENET1K_V1' if pretrained else None)
          # Replace all BatchNorm with GroupNorm for Opacus compatibility
          backbone = replace_bn_with_gn(backbone)
          # Remove final classifier
          self.features = backbone.features
          self.avgpool = backbone.avgpool
          # Feature dim: 576 (MobileNetV3-Small final conv output)
          if freeze_backbone:
              # Freeze first 8 feature blocks
              for i, layer in enumerate(self.features):
                  if i < 8:
                      for param in layer.parameters():
                          param.requires_grad = False

      def forward(self, x):
          x = self.features(x)     # (B, 576, 7, 7)
          x = self.avgpool(x)      # (B, 576, 1, 1)
          return x.flatten(1)      # (B, 576)

  def replace_bn_with_gn(module, num_groups=32):
  """Recursively replace all BatchNorm2d/1d with GroupNorm."""
      for name, child in module.named_children():
          if isinstance(child, (nn.BatchNorm2d, nn.BatchNorm1d)):
              num_channels = child.num_features
              groups = min(num_groups, num_channels)
              # Ensure divisibility
              while num_channels % groups != 0:
                  groups //= 2
              setattr(module, name, nn.GroupNorm(groups, num_channels))
          else:
              replace_bn_with_gn(child, num_groups)
      return module

  class HFL_MM(nn.Module):
  """
  Full HFL-MM model: Dual encoder + Late fusion + Classifier
  """
      def __init__(self, in_channels_a=1, num_classes=4,
                   pretrained=True, freeze_backbone=True):
          super().__init__()
          self.encoder_a = Encoder_A_1DCNN_GRU(
              in_channels=in_channels_a, gru_hidden=128)
          self.encoder_b = Encoder_B_MobileNetV3(
              pretrained=pretrained, freeze_backbone=freeze_backbone)
          # Fusion: 128 + 576 = 704
          self.fusion = nn.Sequential(
              nn.Linear(704, 512),
              nn.LayerNorm(512),
              nn.ReLU(),
              nn.Dropout(0.3),
              nn.Linear(512, 256),
              nn.LayerNorm(256),
              nn.ReLU(),
              nn.Linear(256, num_classes)
          )

      def forward(self, signal, image, mask_a=None, mask_b=None):
          """
          signal: (B, C_in, L) — time-series modality
          image : (B, 3, 224, 224) — image modality
          mask_a: (B,) bool — True if modality A is available
          mask_b: (B,) bool — True if modality B is available
          Graceful degradation: zero-vector for missing modality
          """
          feat_a = self.encoder_a(signal)   # (B, 128)
          feat_b = self.encoder_b(image)    # (B, 576)

          # Apply masks for missing modality handling
          if mask_a is not None:
              feat_a = feat_a * mask_a.float().unsqueeze(1)
          if mask_b is not None:
              feat_b = feat_b * mask_b.float().unsqueeze(1)

          fused = torch.cat([feat_a, feat_b], dim=1)  # (B, 704)
          return self.fusion(fused)                   # (B, num_classes)

  Why Opacus:
    - Official PyTorch DP library (Meta AI)
    - Handles per-sample gradient clipping automatically
    - RDP (Rényi Differential Privacy) accountant built-in
    - Compatible with GRU, Conv1D, Linear (not LSTM due to bug in v1.x)
    - Supports virtual batches (accumulate gradients, clip at micro-batch)

  Setup code:

    from opacus import PrivacyEngine
    from opacus.validators import ModuleValidator

    # Validate model (checks for BN, unsupported layers)
    model = ModuleValidator.fix(model)  # auto-fixes any remaining issues
    errors = ModuleValidator.validate(model, strict=False)
    assert len(errors) == 0, f"Model not Opacus-compatible: {errors}"

    # Optimizer
    optimizer = torch.optim.Adam(model.parameters(), lr=1e-3)

    # Privacy engine
    privacy_engine = PrivacyEngine()
    model, optimizer, train_loader = privacy_engine.make_private_with_epsilon(
        module=model,
        optimizer=optimizer,
        data_loader=train_loader,
        epochs=local_epochs,            # 1 epoch per FL round
        target_epsilon=1.0,
        target_delta=1e-5,
        max_grad_norm=1.0,              # gradient clip norm C
    )
    # Opacus auto-calculates noise_multiplier σ for target ε

    # After each FL round, check privacy budget:
    epsilon = privacy_engine.get_epsilon(delta=1e-5)
    print(f"Privacy budget spent: ε = {epsilon:.3f}")

File: SIMULATORS/model/compression.py

  Why compression AFTER DP noise addition:
    DP noise is added to gradients before compression.
    Compression after DP: gradient is already privacy-safe → compressing
    does not violate DP guarantee (post-processing theorem).
    Compression before DP: would violate privacy (compressed gradients
    reveal more about individual samples).

  Step 1 — Top-k Sparsification with Error Feedback:

    class GradientCompressor:
        def __init__(self, sparsity=0.2):
            self.sparsity = sparsity
            self.residuals = {}   # error feedback per layer

        def compress(self, gradient_dict, round_id):
            compressed = {}
            for name, grad in gradient_dict.items():
                # Add accumulated residual
                if name in self.residuals:
                    grad = grad + self.residuals[name]
                # Top-k selection
                flat = grad.flatten()
                k = max(1, int(len(flat) * self.sparsity))
                values, indices = torch.topk(flat.abs(), k)
                selected_values = flat[indices]
                # Save residual (error feedback for next round)
                residual = grad.clone()
                residual.flatten()[indices] = 0
                self.residuals[name] = residual
                compressed[name] = (indices.cpu(), selected_values.cpu(),
                                    grad.shape)
            return compressed

        def decompress(self, compressed_dict):
            gradients = {}
            for name, (indices, values, shape) in compressed_dict.items():
                flat = torch.zeros(np.prod(shape))
                flat[indices] = values
                gradients[name] = flat.reshape(shape)
            return gradients

  Step 2 — 8-bit Quantization:

    def quantize_8bit(values):
        """Quantize float32 values to uint8."""
        v_min, v_max = values.min(), values.max()
        scale = (v_max - v_min) / 255.0 + 1e-8
        quantized = ((values - v_min) / scale).round().clamp(0, 255).byte()
        return quantized, v_min, scale

    def dequantize_8bit(quantized, v_min, scale):
        """Reconstruct float32 from uint8."""
        return quantized.float() * scale + v_min

  Combined compression ratio:
    Top-k (sparsity=0.2): keep 20% → 5× reduction
    8-bit quantization: float32 → uint8 → 4× reduction
    Combined: 20× reduction in gradient bytes transmitted

  Communication saving from Phase 4:
    Phase 4 used 5× compression → 10 MB per device per round
    Phase 5 uses 20× compression → 2.5 MB per device per round
    NS-3 re-run in Phase 5 will update uplink estimate accordingly.

    import torch
    import torch.onnx
    from onnxruntime.quantization import quantize_dynamic, QuantType
    from onnxruntime.transformers import optimizer

    def export_to_onnx(model, save_path="hfl_mm.onnx"):
        model.eval()
        # Dummy inputs matching model signature
        dummy_signal = torch.randn(1, 1, 1024)     # Encoder A input
        dummy_image  = torch.randn(1, 3, 224, 224)  # Encoder B input

        torch.onnx.export(
            model,
            (dummy_signal, dummy_image),
            save_path,
            opset_version=17,
            input_names=["signal", "image"],
            output_names=["logits"],
            dynamic_axes={
                "signal": {0: "batch_size"},
                "image":  {0: "batch_size"},
                "logits": {0: "batch_size"}
            },
            do_constant_folding=True,
            export_params=True,
            verbose=False
        )
        print(f"FP32 ONNX saved: {save_path}")

    def quantize_int8(onnx_path, output_path="hfl_mm_int8.onnx"):
        quantize_dynamic(
            model_input=onnx_path,
            model_output=output_path,
            weight_type=QuantType.QInt8,
            per_channel=True,
            reduce_range=False,
            optimize_model=True
        )
        print(f"INT8 ONNX saved: {output_path}")

    def benchmark_onnx(model_path, n_runs=1000, warmup=100):
        import onnxruntime as ort
        import numpy as np, time

        sess_options = ort.SessionOptions()
        sess_options.graph_optimization_level = ort.GraphOptimizationLevel.ORT_ENABLE_ALL
        sess_options.intra_op_num_threads = 4

        # Try CUDA first, fallback to CPU
        providers = ["CUDAExecutionProvider", "CPUExecutionProvider"]
        session = ort.InferenceSession(model_path, sess_options,
                                       providers=providers)

        signal_np = np.random.randn(1, 1, 1024).astype(np.float32)
        image_np  = np.random.randn(1, 3, 224, 224).astype(np.float32)

        latencies = []
        for i in range(warmup + n_runs):
            t0 = time.perf_counter()
            _ = session.run(None, {"signal": signal_np, "image": image_np})
            t1 = time.perf_counter()
            if i >= warmup:
                latencies.append((t1 - t0) * 1000)  # ms

        latencies = np.array(latencies)
        print(f"Inference latency (ms):")
        print(f"  Mean: {latencies.mean():.2f}")
        print(f"  P50:  {np.percentile(latencies, 50):.2f}")
        print(f"  P95:  {np.percentile(latencies, 95):.2f}")
        print(f"  P99:  {np.percentile(latencies, 99):.2f}")
        print(f"  Max:  {latencies.max():.2f}")
        return latencies

11.2  PatchTokenizer (unified, handles both modalities)

  class PatchTokenizer(nn.Module):
      """
      Converts signal OR image into token sequence of shape (B, N, D).
      D=128 is shared embedding dim for both modalities.
      """
      def __init__(self, mode="signal", embed_dim=128):
          super().__init__()
          self.mode = mode
          if mode == "signal":
              # Non-overlapping 1D patches: 1024 samples / 64 = 16 patches
              self.proj = nn.Conv1d(1, embed_dim, kernel_size=64, stride=64)
              self.pos_embed = nn.Parameter(torch.zeros(1, 16, embed_dim))
          elif mode == "image":
              # ViT-style 2D patches: 224/16 = 14 → 14×14 = 196 patches
              self.proj    = nn.Conv2d(3, 768, kernel_size=16, stride=16)
              self.compress = nn.Linear(768, embed_dim)
              self.pos_embed = nn.Parameter(torch.zeros(1, 196, embed_dim))
          nn.init.trunc_normal_(self.pos_embed, std=0.02)

      def forward(self, x):
          if self.mode == "signal":
              x = self.proj(x)                    # (B, D, 16)
              x = x.transpose(1, 2)               # (B, 16, D)
              return x + self.pos_embed
          else:
              x = self.proj(x)                    # (B, 768, 14, 14)
              x = x.flatten(2).transpose(1, 2)    # (B, 196, 768)
              x = self.compress(x)                # (B, 196, D)
              return x + self.pos_embed

11.3  CMGA Block (Cross-Modal Gated Attention)

  class CMGA_Block(nn.Module):
      """
      One CMGA block. Stack 3 of these for the full encoder.
      z_a: (B, N_a+1, D)  — signal tokens + CLS
      z_b: (B, N_b+1, D)  — image tokens + CLS
      GroupNorm used instead of LayerNorm for Opacus compatibility
      on the projection weights (LayerNorm on sequence dim is fine).
      """
      def __init__(self, dim=128, n_heads=8, dropout=0.1, ffn_mult=4):
          super().__init__()
          # Within-modality self-attention
          self.self_a = nn.MultiheadAttention(dim, n_heads,
                            dropout=dropout, batch_first=True)
          self.self_b = nn.MultiheadAttention(dim, n_heads,
                            dropout=dropout, batch_first=True)
          # Cross-modal attention: A→B and B→A
          self.cross_ab = nn.MultiheadAttention(dim, n_heads,
                              dropout=dropout, batch_first=True)
          self.cross_ba = nn.MultiheadAttention(dim, n_heads,
                              dropout=dropout, batch_first=True)
          # Learned relevance gates
          self.gate_a = nn.Linear(dim, dim)
          self.gate_b = nn.Linear(dim, dim)
          # Normalisation + FFN
          self.norm = nn.ModuleList([nn.LayerNorm(dim) for _ in range(4)])
          self.ffn_a = nn.Sequential(
              nn.Linear(dim, dim * ffn_mult), nn.GELU(),
              nn.Dropout(dropout), nn.Linear(dim * ffn_mult, dim))
          self.ffn_b = nn.Sequential(
              nn.Linear(dim, dim * ffn_mult), nn.GELU(),
              nn.Dropout(dropout), nn.Linear(dim * ffn_mult, dim))
          self.drop = nn.Dropout(dropout)

      def forward(self, z_a, z_b):
          # --- Within-modal self-attention ---
          sa, _ = self.self_a(z_a, z_a, z_a)
          z_a   = self.norm[0](z_a + self.drop(sa))
          sb, _ = self.self_b(z_b, z_b, z_b)
          z_b   = self.norm[1](z_b + self.drop(sb))

          # --- Cross-modal attention with gate ---
          # Signal attends to image (A queries B)
          ca, _ = self.cross_ab(z_a, z_b, z_b)
          g_a   = torch.sigmoid(self.gate_a(ca))
          z_a   = self.norm[2](z_a + self.drop(g_a * ca))
          z_a   = z_a + self.ffn_a(z_a)

          # Image attends to signal (B queries A)
          cb, _ = self.cross_ba(z_b, z_a, z_a)
          g_b   = torch.sigmoid(self.gate_b(cb))
          z_b   = self.norm[3](z_b + self.drop(g_b * cb))
          z_b   = z_b + self.ffn_b(z_b)

          return z_a, z_b

11.4  PersonalAdapter (pFAL — local only, never shared)

  class PersonalAdapter(nn.Module):
      """
      Bottleneck adapter: 128 → 64 → 128 with residual connection.
      Total params: 128×64 + 64 + 64×128 + 128 = 16,576 ≈ 16K params.
      Never transmitted to edge or cloud.
      Trained with standard SGD — no DP required since weights stay local.
      """
      def __init__(self, dim=128, bottleneck=64):
          super().__init__()
          self.down = nn.Linear(dim, bottleneck)
          self.up   = nn.Linear(bottleneck, dim)
          self.act  = nn.GELU()
          # Init to near-identity: up weights small so adapter starts neutral
          nn.init.zeros_(self.up.weight)
          nn.init.zeros_(self.up.bias)

      def forward(self, x):
          return x + self.up(self.act(self.down(x)))  # residual

11.5  PHANTOM_FL — Full Model Class

  class PHANTOM_FL(nn.Module):
      """
      Full PHANTOM-FL model.
      Backbone (federated): PatchTokenizers + CMGA blocks + fusion FC
      Adapter  (local-only): PersonalAdapter
      Head     (federated): Linear classifier
      """
      def __init__(self, num_classes=4, n_cmga_blocks=3,
                   dim=128, n_heads=8, dropout=0.1):
          super().__init__()
          # Patch tokenizers (one per modality)
          self.tok_a = PatchTokenizer(mode="signal", embed_dim=dim)
          self.tok_b = PatchTokenizer(mode="image",  embed_dim=dim)
          # CLS tokens (one per modality, learnable)
          self.cls_a = nn.Parameter(torch.zeros(1, 1, dim))
          self.cls_b = nn.Parameter(torch.zeros(1, 1, dim))
          nn.init.trunc_normal_(self.cls_a, std=0.02)
          nn.init.trunc_normal_(self.cls_b, std=0.02)
          # CMGA encoder blocks
          self.cmga_blocks = nn.ModuleList(
              [CMGA_Block(dim, n_heads, dropout) for _ in range(n_cmga_blocks)])
          # Fusion
          self.fusion_norm = nn.LayerNorm(dim * 2)
          self.fusion_fc   = nn.Sequential(
              nn.Linear(dim * 2, dim), nn.GELU(), nn.Dropout(dropout))
          # Personal adapter (local only — call freeze_adapter() to protect)
          self.adapter = PersonalAdapter(dim)
          # Classification head
          self.head = nn.Linear(dim, num_classes)

      def forward(self, signal, image):
          """
          signal: (B, 1, 1024)    — raw vibration / wearable signal
          image : (B, 3, 224, 224) — spectrogram / X-ray image
          Returns logits: (B, num_classes)
          """
          B = signal.size(0)
          # Tokenize
          z_a = self.tok_a(signal)              # (B, 16, D)
          z_b = self.tok_b(image)               # (B, 196, D)
          # Prepend CLS tokens
          z_a = torch.cat([self.cls_a.expand(B,-1,-1), z_a], dim=1)  # (B,17,D)
          z_b = torch.cat([self.cls_b.expand(B,-1,-1), z_b], dim=1)  # (B,197,D)
          # CMGA encoder
          for block in self.cmga_blocks:
              z_a, z_b = block(z_a, z_b)
          # Extract CLS token representations
          feat_a = z_a[:, 0, :]                 # (B, D)
          feat_b = z_b[:, 0, :]                 # (B, D)
          # Fuse
          fused = self.fusion_norm(
              torch.cat([feat_a, feat_b], dim=1))  # (B, 2D)
          fused = self.fusion_fc(fused)             # (B, D)
          # Personal adapter
          fused = self.adapter(fused)               # (B, D)
          return self.head(fused)                   # (B, num_classes)

      def backbone_parameters(self):
          """Return only federated params (exclude adapter)."""
          adapter_ids = {id(p) for p in self.adapter.parameters()}
          return [p for p in self.parameters() if id(p) not in adapter_ids]

      def adapter_parameters(self):
          """Return only local-only adapter params."""
          return list(self.adapter.parameters())

12.1  Sensitivity-Adaptive DP (SADP) — Innovation I2

  Problem with HFL-MM: uniform σ=1.1 on every layer wastes budget.
  Early conv layers detect edges/frequencies (low sensitivity).
  Classifier head maps features to private labels (high sensitivity).
  Same noise level for both is suboptimal.

  SADP engine computes per-layer gradient norm BEFORE clipping,
  then scales noise proportionally (higher sensitivity → more noise):

  class SADPEngine:
      def __init__(self, base_sigma=1.1, alpha=0.5, min_ratio=0.1):
          self.base_sigma = base_sigma
          self.alpha      = alpha        # controls steepness of scaling
          self.min_ratio  = min_ratio    # floor: at least 10% base noise
          self.layer_sigmas = {}

      def calibrate(self, model):
          """
          Call AFTER backward() but BEFORE optimizer.step().
          Computes per-layer gradient sensitivity and sets σ_l.
          """
          sensitivities = {}
          for name, param in model.named_parameters():
              if param.grad is not None:
                  sensitivities[name] = param.grad.detach().norm().item()
          max_s = max(sensitivities.values()) + 1e-8
          for name, s in sensitivities.items():
              ratio = max(self.min_ratio, (s / max_s) ** self.alpha)
              self.layer_sigmas[name] = self.base_sigma * ratio
          return self.layer_sigmas

      def inject_noise(self, model, clip_norm=1.0):
          """Add adaptive Gaussian noise to each gradient."""
          for name, param in model.named_parameters():
              if param.grad is not None:
                  sigma = self.layer_sigmas.get(name, self.base_sigma)
                  noise = torch.randn_like(param.grad) * sigma * clip_norm
                  param.grad.add_(noise)

  NOTE: SADP is an ADDITION to Opacus, not a replacement.
  Use Opacus for per-sample gradient clipping (mandatory for DP).
  Use SADP for the noise injection step (override Opacus default noise):

    # In training loop:
    optimizer.zero_grad()
    loss.backward()
    # Per-sample clip (Opacus handles automatically via hooks)
    sadp_engine.calibrate(model)      # compute σ_l per layer
    sadp_engine.inject_noise(model)   # add adaptive noise
    optimizer.step()                  # update params

  Privacy accounting: SADP uses RDP composition per-layer.
  Since σ_l ≤ σ_base for all layers, total ε ≤ ε from uniform σ_base.
  Conservative bound: use Opacus accountant with base_sigma (safe).

12.2  FedKD-E — Federated Knowledge Distillation at Edge (Innovation I4)

  Problem with FedAvg: gradient averaging conflicts when devices have
  very different class distributions (non-IID, α=0.5 Dirichlet).
  Solution: edge distills an ensemble of received local models into
  the edge model using 500 unlabeled proxy samples.

  Proxy dataset D_proxy:
    NOT from any device — no privacy concern.
    Options: synthetic (random Gaussian), or public domain samples
    (e.g., 500 random CWRU samples from a public online source,
     different machine/operating conditions than training set).

  Distillation loop at edge server:

  def edge_distill(local_models, weights, edge_model,
                   proxy_loader, steps=50, T=3.0, lr=1e-3):
      """
      local_models : list of received device model state_dicts
      weights      : staleness-adjusted weight per model (ASWA)
      edge_model   : edge student model (same PHANTOM_FL architecture)
      proxy_loader : DataLoader over D_proxy (no labels needed)
      T            : temperature for soft labels
      """
      edge_model.train()
      optimizer = torch.optim.Adam(edge_model.parameters(), lr=lr)
      kl_loss   = nn.KLDivLoss(reduction="batchmean")

      # Load local model weights into temporary model instances
      teachers = []
      for state_dict in local_models:
          teacher = PHANTOM_FL(num_classes=NUM_CLASSES)
          teacher.load_state_dict(state_dict)
          teacher.eval()
          teachers.append(teacher)

      for step, (signal, image) in enumerate(proxy_loader):
          if step >= steps:
              break
          # Ensemble teacher: weighted average of softmax outputs
          with torch.no_grad():
              teacher_probs = sum(
                  w * torch.softmax(t(signal, image) / T, dim=-1)
                  for t, w in zip(teachers, weights)
              ) / sum(weights)
          # Student distillation
          student_logits = edge_model(signal, image)
          student_log_probs = torch.log_softmax(student_logits / T, dim=-1)
          loss = (T ** 2) * kl_loss(student_log_probs, teacher_probs)
          optimizer.zero_grad()
          loss.backward()
          optimizer.step()

      return edge_model.state_dict()

12.3  ASWA — Asynchronous Staleness-Weighted Aggregation (Innovation I3)

  τ_min = floor(0.7 × cluster_size)  # wait for 70%, not 100%

  def aswa_weights(arrivals, current_time):
      """
      arrivals: list of (device_id, state_dict, send_time, n_samples)
      Returns: weights for FedKD-E (or FedAvg fallback)
      """
      weights = []
      for _, _, t_send, n_i in arrivals:
          staleness = max(0.0, current_time - t_send)   # seconds
          s_i = 1.0 / (1.0 + staleness)                 # staleness factor
          weights.append(n_i * s_i)
      total = sum(weights)
      return [w / total for w in weights]                # normalised

  Edge aggregation trigger:
    While True:
        if len(arrival_queue) >= tau_min:
            current_time = time.time()
            weights = aswa_weights(arrival_queue, current_time)
            edge_model = edge_distill(models, weights, edge_model, proxy_loader)
            break
        if time.time() - round_start > TIMEOUT_SECONDS:
            # Force aggregate with whoever arrived — avoid deadlock
            weights = aswa_weights(arrival_queue, time.time())
            edge_model = edge_distill(models, weights, edge_model, proxy_loader)
            break
